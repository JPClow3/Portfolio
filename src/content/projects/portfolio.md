## Overview

A modern, performant portfolio website showcasing my work as a Front-End Developer. Built with cutting-edge web technologies and best practices for accessibility and user experience.

## Features

- **Islands Architecture** - Static-first with hydration only where needed
- **Dark/Light Mode** - System preference detection with smooth transitions
- **3D Hero Scene** - Interactive Three.js background with particle effects
- **Scroll Animations** - IntersectionObserver-based reveal animations
- **Accessibility** - Skip links, ARIA labels, reduced-motion support
- **SEO Optimized** - JSON-LD structured data, Open Graph tags
- **Responsive Design** - Mobile-first approach with fluid typography

## Tech Stack

- **Astro 5** - Static site generation with islands
- **Svelte 5** - Interactive UI components (runes syntax)
- **TypeScript** - Type safety throughout
- **Tailwind CSS v4** - Utility-first styling
- **Three.js** - 3D WebGL effects
- **Railway** - Static deployment
