## Overview

A lightweight, extensible framework built with TypeScript for automated API testing. Designed to validate RESTful API endpoints with comprehensive test coverage.

## Features

- **Request Testing** - Support for GET, POST, PUT, DELETE methods
- **Response Validation** - Automatic status code and body verification
- **TypeScript** - Full type safety for test definitions
- **Extensible** - Easy to add new test cases and validators

## Test Capabilities

- Status code validation
- Response body schema validation
- Header verification
- Response time assertions
- Error handling tests
